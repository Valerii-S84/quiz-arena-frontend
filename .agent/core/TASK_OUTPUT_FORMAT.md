# TASK_OUTPUT_FORMAT

Фінальна відповідь агента має бути тільки в цій структурі:
```
Статус: Done | Partial | Incomplete

Scope:
...

Створені/оновлені файли:
- ...

Що зроблено:
- ...

Що НЕ робилося:
- ...

Перевірка:
- ...

Недоведено / ризики:
- ...
```

## Правила формату

- Пиши коротко.
- Пиши тільки по факту задачі.
- Не додавай преамбулу.
- Не додавай поради без прямого запиту.
- Якщо пункт порожній, пиши `- Немає`.
- `Partial` або `Incomplete` вимагають заповненого поля `Недоведено / ризики`.
