# AGENTS.md

Цей файл є compatibility wrapper.

Активна точка входу для агентських правил:

- `.agent/AGENTS.md`

Правила використання:

1. Якщо агент стартує з кореня репозиторію, він спочатку читає
   `.agent/AGENTS.md`.
2. Єдиним джерелом істини вважаються документи в `.agent/core/`
   і `.agent/project/`.
3. Кореневі legacy-нормативні файли не є активними правилами.
